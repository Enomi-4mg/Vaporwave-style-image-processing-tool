# Vaporwave Generator (ヴェイパーウェイヴ・ジェネレーター)

これは、画像にヴェイパーウェイヴ風のエフェクトを適用するopenFrameworks製のアプリケーションです。カラーマッピング、グリッチ、スキャンライン、ノイズといったエフェクトをリアルタイムで適用できます。

## 主な機能

-   **カラーマッピング**: ルックアップテーブル（LUT）を使用して、画像のカラーパレットをヴェイパーウェイヴ特有の美的感覚に調整します。
-   **グリッチエフェクト**: 画像に水平方向のグリッチ（画面の乱れ）を生成します。
-   **スキャンライン**: CRTスクリーンのような走査線をオーバーレイします。
-   **ノイズ**: Lo-fiな質感を与えるためのランダムなノイズを追加します。
-   **リアルタイムコントロール**: すべてのパラメータはGUIを通じてリアルタイムで調整可能です。
-   **ドラッグ＆ドロップ**: 自分の画像をアプリケーションウィンドウにドラッグ＆ドロップして読み込ませることができます。

## 使い方

1.  アプリケーションを起動します。
2.  デフォルトでサンプル画像（`pic.png`）が読み込まれます。
3.  左側に表示されるGUIを使用して、さまざまなエフェクトのON/OFFやパラメータの調整を行います。
4.  ウィンドウに手持ちの画像ファイルをドラッグ＆ドロップすると、その画像にエフェクトを適用できます。
5.  `s`キーを押すと、現在表示されているエフェクト適用後の画像を保存できます。保存されたファイルは`bin/data`ディレクトリに`saved_image.png`という名前で保存されます。

## 操作方法

-   **GUI**: すべてのエフェクトパラメータを操作します。
-   **ドラッグ＆ドロップ**: 新しい画像を読み込みます。
-   **'s' キー**: 現在のキャンバスを`saved_image.png`として保存します。

## パラメータ

GUIを通じて以下のパラメータをコントロールできます。

-   **Color Mapping (カラーマッピング)**
    -   `Deep Blue`, `Magenta`, `Cyan`, `White`: グラデーションマップの基準となる色を定義します。
    -   `Thresh 1`, `Thresh 2`: 色の移り変わりの閾値を調整します。
    -   `Apply Effect`: カラーマッピングエフェクトのON/OFFを切り替えます。
-   **Glitch Settings (グリッチ設定)**
    -   `Glitch Count`: グリッチの線の数。
    -   `Max Offset`: グリッチの最大の水平方向のズレ幅。
    -   `Min Height`, `Max Height`: 各グリッチの線の高さの範囲。
    -   `Apply Glitch`: グリッチエフェクトのON/OFFを切り替えます。
-   **Texture Settings (テクスチャ設定)**
    -   `Scanline Density`: スキャンラインの間隔。
    -   `Scanline Alpha`: スキャンラインの不透明度。
    -   `Apply Scanlines`: スキャンラインエフェクトのON/OFFを切り替えます。
    -   `Noise Count`: ノイズの粒の数。
    -   `Noise Alpha`: ノイズの不透明度。
    -   `Apply Noise`: ノイズエフェクトのON/OFFを切り替えます。

## ビルド方法

このプロジェクトはopenFrameworksで開発されています。環境構築やプロジェクトのビルド方法については、openFrameworksの公式ドキュメントを参照してください。

1.  openFrameworks (v0.12.0以降を推奨)をダウンロードし、セットアップします。
2.  このプロジェクトフォルダを `of_v0.12.0_vs_64/apps/myApps/` の中に配置します。
3.  Visual Studioでソリューションファイル（`.sln`）を開き、コンパイルします。
